# Mathcode Academy

Mathcode Academy is a competitive programming platform inspired by Codeforces.

## Features
- Authentication system
- User and admin roles
- Problem management
- Contest participation
- Submission and verdict system
- Leaderboard
- Judge service

## Tech Stack
- React.js
- FastAPI
- SQLite
- SQLAlchemy
- Vite
