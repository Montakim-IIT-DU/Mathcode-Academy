# Mathcode Academy - Project Overview

Mathcode Academy is a Codeforces-inspired competitive programming platform built with React.js and FastAPI.

## Main Features
- User registration and login
- Admin dashboard
- Problem creation and management
- Contest creation and participation
- Code submission system
- Leaderboard system
- Judge service simulation

## Technology Stack
- Frontend: React.js + Vite
- Backend: FastAPI + SQLite + SQLAlchemy
- Judge: FastAPI-based simulated judge service
